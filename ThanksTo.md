# Other Software used by Lino #

  * [Python](http://www.python.org)
  * [Django](http://www.djangoproject.com)
  * [ExtJS](http://www.extjs.com/products/extjs)
  * Dirk Holtwick's [xhtml2pdf module](http://www.xhtml2pdf.com).
