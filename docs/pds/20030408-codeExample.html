The following function shows how to insert code examples along with their result in a document. 

def codeExample(txt):
   beginTable("ll",width=getTextWidth()*0.9,
              model=stylesheet.DefaultTable)
   setFeeder('plain')
   pre(txt,CodeExampleStyle)
   endCell()
   setFeeder('xml')
   exec txt in globals()
   endTable()
