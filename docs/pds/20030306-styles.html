Playing with styles

Paragraph styles can specify (or not) the following paragraph attributes

  • spaceBefore (0);
  • fontName ('Times-Roman');
  • textColor (Color(0.000000,0.000000,0.000000));
  • keepWithNext (False);
  • leading (12);
  • backColor (None);
  • bulletIndent (0);
  • leftIndent (0);
  • bulletFontSize (10);
  • rightIndent (0);
  • fontSize (10);
  • spaceAfter (0);
  • firstLineIndent (0);
  • wrap (True);
  • bulletFontName ('Times-Roman');
  • allowSplitting (True);
  • alignment (0);

Here is a list of the currently defined paragraph styles:
