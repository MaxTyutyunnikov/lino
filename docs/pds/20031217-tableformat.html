Formatting tables

Formatting tables

pds provides a simplified interface to reportlab's table formatting model. With pds you specify only the command and its parameters, while the coordinates of the cells to be formatted depend on which formatXxx() function you use:

  • formatTable() applies to all the cells of the table's data area
  • formatHeader() applies to all the cells of the table's header
  • formatRow(), formatColumn(), formatCell() to the current row, column or cell.

See reportlab's userguide.pds for details about the formatting commands.

Here is a list of allowed commands: BACKGROUND, GRID, BOX, OUTLINE, INNERGRID, LINEBELOW, LINEABOVE, LINEBEFORE, LINEAFTER, FONT, FONTNAME, SIZE, LEADING, TEXTCOLOR, ALIGN, VALIGN, LEFTPADDING, RIGHTPADDING, TOPPADDING, BOTTOMPADDING
