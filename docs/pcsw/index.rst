Lino for Belgian Public Centres for Social Welfare (Lino-PCSW)
==============================================================

Overview
--------
Lino-PCSW is the first Lino application being used in the real world.
It is currently a part of the Lino project, 
but will probably split off soon to be maintained by an
independant organization. 

It is a tool for social assistants that help receivers 
of public aid to find a suitable job or education, 
adapted to Belgian *Public Centres for Social Welfare* 
(Centres Publics d'Action Sociale).
