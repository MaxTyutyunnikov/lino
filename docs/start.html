Getting started with Lino

The "visible" part of Lino is a collection of standalone scripts.

Currently I can only give the following instructions:

  • Install Lino
  • run the test suites
  • look at the source code.

The source tree has the following main subdirectories

  • tests : test suites. Currently the best place to start.
  • src : the Python source files.
  • docs : documentation. Currently this contains the same as the homepage.

You can also skip installation an browse the CVS repository to have a look at the code.
