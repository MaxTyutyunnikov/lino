usage: lino prn2pdf [options] FILE

where FILE is the file to be converted to a pdf file. It may contain plain text
and simple formatting printer control sequences.
options:
  -h, --help            show this help message and exit
  -lLOGFILE, --logfile=LOGFILE
                        log a report to FILE
  -v, --verbose         increase verbosity
  -q, --quiet           decrease verbosity
  -b, --batch           not interactive (don't ask anything)
  -oOUTFILE, --output=OUTFILE
                        write to OUTFILE rather than FILE.pdf
