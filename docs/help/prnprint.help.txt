usage: lino prnprint [options] FILE [FILE ...]

where FILE is a plain text file to be printed on the Default Printer. It must
be in OEM charset and may contain simple formatting printer control sequences,
see http://lino.berlios.de/prn2pdf.html
options:
  -h, --help            show this help message and exit
  -v, --verbose         increase verbosity
  -q, --quiet           decrease verbosity
  -b, --batch           not interactive (don't ask anything)
  -lLOGFILE, --logfile=LOGFILE
                        log a report to FILE
  -pPRINTERNAME, --printer=PRINTERNAME
                        print on PRINTERNAME rather than on Default Printer.
  -cCOPIES, --copies=COPIES
                        print NUM copies.
  -oSPOOLFILE, --output=SPOOLFILE
                        write to SPOOLFILE rather than really printing.
  -u, --useWorldTransform
                        use SetWorldTransform() to implement landscape
