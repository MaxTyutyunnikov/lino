    LINOLIB_MESSAGES = set()
    tpl = CheetahTemplate(...)

        LINOLIB_MESSAGES.add(s)
        return _(s)
    tpl._ = mytranslate

  Datei :file:`lino.ui.extjs.dummy_messages.py`.
