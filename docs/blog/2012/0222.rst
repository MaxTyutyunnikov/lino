PUT eines Uploads geht nicht?

Klick auf "Startseite" bei leerer history funktionierte nicht, 
wenn außerdem auch ROOT_URL leer war. Also `location.replace('')` 
bewirkt nichts. Muss dann `location.replace('/')` heißen.
