Mitmus
------

.. vocabulary::
      père 
      pullover 
      oncle
      manteau
      
.. question::
    mitmus = ainsus + **-s** : 
    le père → les pères, 
    le pullover → les pullovers, 
    l'oncle → les oncles
    (erandid: la → les, le manteau → les manteaux).
